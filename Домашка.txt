1.Перемещение по папкам- Команда cd выводит название текущей папки. Перейти в папку внутри текущего каталога.
cd Desktop/FE-24.03/Githab/enter

Перейти на одну папку вверх ../

Elzar@LAPTOP-S7VUVH5Q MINGW64 ~/Desktop/FE-24.03 (master)
$ ../ переходит в папку Desktop.

2. Просмотр списка файлов в директории осуществляется через команды ls.
01_absolute\ 'CSS grid'/   Flexbox2/

3. Удаление файлов и папок через команду rm -rf. 

4. Файлы создаются через команду touch. Папки через mkdir.

5. Git репозиторий- это хранилище всех изменений. 
https://github.com/elzarsadybekov/first-repository

6.Создание коммитов: После инициализации (git init) набираем команду git status, за тем git add .
Файлы или папки подгараются зеленым цветом и эти файлы и папки находятся в stage changes. После команды git commit -m "First commit" файлы и папки исчезают и файлы готовы к отправке на удаленный сервер.

7. Прпосмотр коммитов в репозитории. Пожно отправить ссылку.  https://github.com/elzarsadybekov/first-repository
